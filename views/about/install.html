Install

Get the module

$ [sudo] npm install -g express-admin

or

$ npm install express-admin

Create a Project expand

Depending on how you installed it creating a project is as follows

$ admin path/to/config

or

$ node path/to/express-admin/app.js path/to/config

Either way the config path should exist. If there isn't config files in it already you will be prompted to add some required information for your database, server port and admin account.

At this point you can navigate to localhost:[specified port] and see your admin up and running. The next thing is to configure it.

The application settings are stored inside the config.json file. The mysql key accepts any option from the node-mysql's connection options.

{
    "mysql": {
        "database": "express-admin-examples",
        "user": "liolio",
        "password": "karamba"
    },
    "server": {
        "port": 3000
    },
    "app": {
        "layouts": true,
        "themes": true,
        "languages": true
    }
}
