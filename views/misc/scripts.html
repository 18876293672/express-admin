Debug

Debug the admin app as usual

$ admin --debug-brk path/to/project/config/

or

$ node --debug-brk path/to/express-admin/app path/to/project/config/

Forever

Run an admin instance continuously in the background using forever

$ forever start path/to/express-admin/app path/to/project/config/

Supervisor

Auto reload your project while configuring it using supervisor.

Suppose you create the folder for your project's config like this.

$ mkdir project

Then while you're still there you can use supervisor to watch your project folder, specifying that the extensions are json like this.

$ supervisor -w project -e json -- path/to/express-admin/app -v project/

Here the -v flag that you're passing to the Express Admin app means that you are running the admin in development mode. In this mode the authentication and the database operations are disabled.

Make sure you take a look at the Introductory Screencast about Express Admin.
